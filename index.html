VilvBot

Официальный поисковый краулер для VilvSearch

О VilvBot

VilvBot — это официальный краулер для индексирования страниц в поисковой системе VilvSearch. Он собирает данные о контенте сайтов, чтобы улучшать результаты поиска, и работает с уважением к правилам сайтов и ограничениям скорости запросов.

Правила работы

  • Следует правилам robots.txt сайтов
  • Не превышает допустимый rate-limit: 1–2 запроса в секунду
  • Использует User-Agent: VilvBot/1.0 (+https://vilvbot.github.io)

Политика уважения

VilvBot индексирует сайты исключительно для улучшения поиска. Никогда не перегружает серверы и не используется для спама или вредоносной активности.

Инструкция для вебмастеров

Чтобы разрешить или запретить доступ VilvBot к вашему сайту, используйте один из способов:

1. Через robots.txt

Запретить доступ к всему сайту:

User-agent: VilvBot
Disallow: /

Разрешить доступ ко всему сайту:

User-agent: VilvBot
Allow: /

2. Через Cloudflare

  1. Перейдите в панель сайта на Cloudflare → Firewall → Tools → User Agent Blocking
  2. Добавьте правило:
    • User-Agent: VilvBot
    • Action: Block (для запрета) или Allow (для разрешения)

После этого Cloudflare будет блокировать или разрешать все запросы с VilvBot.
